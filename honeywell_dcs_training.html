// Step-by-Step Guide
Configuration Studio Walkthrough

From launch to full server configuration — every step documented and visualised.

01
Launch Configuration Studio
Navigate to Start → All Programs → Honeywell Experion PKS → Configuration Studio. It can be run locally or remotely from any client.
02
Connect to Experion Server
Enter the server name or IP address. Configuration Studio connects to the ERDB (Experion Relational Database) via the FTE network.
03
Explore the Enterprise Model
Set up the Asset hierarchy in the Enterprise Model Builder. Define Areas, Units, and Equipment with alarm groups and data paths.
04
Configure Flex, Console & Extension Stations
Add and configure Experion Station types — Console, Console Extension, Flex, and Report printers — with appropriate settings.
05
Add Controllers & I/O Modules
Insert the C300 controller in Control Builder, load CPU and interface modules, assign Series C I/O chassis and channel addresses.
06
Create SCADA Points & Algorithms
Define SCADA channels and points with appropriate engineering units, alarm limits, scaling, and SCADA Point Algorithms.
07
Load & Validate Configuration
Perform a data load to the ERDB and to target controllers. Validate the configuration, monitor load progress, and resolve any errors.
08
Verify on Experion Station
Confirm assets, points, alarms, and trends appear correctly on the operator station. Perform a live I/O check and alarm verification.
